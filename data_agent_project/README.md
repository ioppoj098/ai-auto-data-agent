# AI自动数据分析Agent（火山方舟大模型）
## 项目介绍
基于Python + 火山方舟大模型搭建自动化数据分析工具，输入自然语言需求，自动生成、清洗、执行Pandas分析代码，输出统计结果与Matplotlib可视化图表，全程记录运行日志。
### 核心功能
1. 读取零售销售数据集sales_data.csv
2. 自然语言自动转换数据分析Python代码
3. 正则过滤模型冗余文本，提取可执行代码
4. 分层异常捕获：API接口报错、代码运行错误分开记录
5. 自动生成高清可视化柱状图
6. 带时间戳日志留存全流程运行记录
